# Tesla Stock Price Prediction using Deep Learning

## 📌 Project Overview
This project focuses on predicting stock prices of Tesla using deep learning models such as SimpleRNN and LSTM. The goal is to analyze historical stock data and forecast future price movements.

## 🏢 Company
Tesla (Electric Vehicle & Clean Energy Company)

## 🎯 Objectives
- Analyze historical stock data
- Perform data cleaning and preprocessing
- Build deep learning models (SimpleRNN & LSTM)
- Predict future stock prices
- Compare model performance

## 📊 Dataset
- File: TSLA.csv
- Features:
  - Date
  - Open
  - High
  - Low
  - Close
  - Adj Close
  - Volume

## ⚙️ Technologies Used
- Python
- Pandas, NumPy
- Matplotlib
- Scikit-learn
- TensorFlow / Keras
- Google Colab / Jupyter Notebook

## 🔄 Project Workflow
1. Data Cleaning
2. Data Preprocessing
3. Exploratory Data Analysis (EDA)
4. Data Visualization
5. Feature Engineering
6. Model Building (RNN & LSTM)
7. Model Evaluation

## 🤖 Models Used
### SimpleRNN
- Captures short-term dependencies

### LSTM
- Captures long-term dependencies
- Performs better than RNN

## 📈 Results
- LSTM model achieved better accuracy
- Lower Mean Squared Error (MSE)
- Predictions closer to actual stock prices

## 🚀 Future Improvements
- Add news sentiment analysis (NLP)
- Use advanced models (GRU, Transformer)
- Include macroeconomic indicators

## 📌 Conclusion
LSTM is more effective than SimpleRNN for time-series forecasting due to its ability to capture long-term dependencies.
