# Project2ML-Tesla-Stock-Price-PredictionDS

# Tesla Stock Price Prediction using Deep Learning

## 📌 Overview
This project aims to predict stock prices of Tesla using deep learning models such as SimpleRNN and LSTM. It includes both implementation (notebook) and detailed report documentation.

## 🎯 Objectives
- Analyze Tesla stock data
- Perform preprocessing and feature engineering
- Build and compare RNN & LSTM models
- Predict future stock prices

## 📊 Dataset
- TSLA.csv (Tesla stock data)
- Features include Open, Close, High, Low, Volume, etc.

## ⚙️ Tech Stack
- Python
- Pandas, NumPy
- Matplotlib
- Scikit-learn
- TensorFlow / Keras

## 🔄 Workflow
1. Data Cleaning
2. Data Preprocessing
3. EDA & Visualization
4. Feature Engineering
5. Model Building
6. Model Evaluation

## 🤖 Models
### SimpleRNN
- Good for short-term patterns

### LSTM
- Better for long-term dependencies
- More accurate predictions

## 📈 Results
- LSTM outperformed SimpleRNN
- Lower MSE
- Better prediction accuracy

## 📄 Report Includes
- Complete project explanation
- Methodology and analysis
- Model comparison
- Insights and conclusion

## 🚀 Future Scope
- Add NLP-based sentiment analysis
- Use advanced models (GRU, Transformer)
- Improve feature engineering

## 📌 Conclusion
This project shows that LSTM is highly effective for stock price prediction and can be used in real-world financial applications.
